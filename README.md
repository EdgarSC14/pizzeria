# Pizza Express - Sitio Web de Pizzería

Este es un sitio web moderno y responsivo para una pizzería, desarrollado con HTML, CSS y JavaScript puro.

## Características

- Diseño moderno y responsivo
- Sección de información de la empresa
- Organigrama de la pizzería
- Menú de productos con carrito de compras
- Múltiples métodos de pago
- Generación de PDF para los pedidos
- Navegación suave entre secciones

## Tecnologías Utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla)
- jsPDF para la generación de PDFs
- Font Awesome para iconos

## Cómo Usar

1. Clona este repositorio o descarga los archivos
2. Abre el archivo `index.html` en tu navegador web
3. Navega por las diferentes secciones usando el menú de navegación
4. Para realizar un pedido:
   - Selecciona las pizzas que deseas del menú
   - Revisa tu carrito de compras
   - Haz clic en "Imprimir Pedido" para generar el PDF

## Estructura del Proyecto

```
pizzeria/
├── index.html          # Página principal
├── styles.css          # Estilos CSS
├── script.js           # Funcionalidad JavaScript
└── README.md          # Documentación
```

## Características del Carrito de Compras

- Agregar productos al carrito
- Ver el total de la compra
- Generar PDF del pedido
- Actualización en tiempo real del carrito

## Personalización

Puedes personalizar el sitio web modificando:

- Los colores en `styles.css` (busca las variables de color)
- Las imágenes de las pizzas en `index.html`
- Los precios y descripciones de los productos
- La información de contacto en el footer

## Requisitos

- Navegador web moderno
- Conexión a internet (para cargar Font Awesome y jsPDF)
